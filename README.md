# Repositório do curso Santander 2025 - Ciência de Dados com Python

## Desafio - Explorando IA Generativa em um Pipeline de ETL com Python

O objetivo deste desafio é replicar o projeto prático, explorando os conceitos de Ciência de Dados e Python, tendo como foco principal aprender o fluxo ETL (Extração, Transformação e Carregamento) e entender como os dados fluem de uma etapa para a outra.
